# Support ChatAPP

Esta aplicación es para explicar cómo usar React Router tanto como librería y como Framework.

## Dev

1. Clonar repositorio
2. Ejecutar `npm install`
3. Ejecutar `npm run dev`

## Tecnologías usadas

- React
- React Router
- Vite
- TailwindCSS
- ShadcnUI
- Lucide Icons
- V0
- TypeScript
